# AI-Finance-TinyGPT
## 1. 프로젝트 개요
본 프로젝트는 Character-level(글자단위) 기반의 소형 GPT 모델을 학습시켜 텍스트 생성의 원리를 이해하고 검증하는 것을 목표로 한다.
기본 제공된 소설 데이터셋을 넘어 애덤스미스의 『국부론 (The Wealth of Nations)』을 학습 데이터로 선정하여 인공지능이 전문적인 학술 어휘와 문체를 모방할 수 있는지 실험했다.
- 데이터 출처 : 프로젝트 구텐베르크 (Project Gutenberg)
- 데이터셋: 애덤 스미스, 『국부론 (The Wealth of Nations)』 순수 텍스트(Plain Text)

## 2. 하이퍼파라미터 튜닝 및 최적화 
국부론 데이터에 맞춰 모델의 성능을 극대화 하기 위해 코드 수정 작업을 진행하였다. 
- Epoch 증가 (TOTAL_EPOCHS = 2000): 복잡하고 긴 문장이 많은 경제학 서적의 특성을 반영하여, 모델의 데이터 패턴을 충분히 학습할 수 있도록 기존 ??회였던 에포크를 2000회로 상향 조정했다.
- 초기 입력값 변경 (start_str = "market "): 학습 완료 후 텍스트를 생성할 때, ???라는 학습되지 않은 단어 대신 경제학 서적에 가장 빈번하게 등장하는 'market'을 시드 단어로 주입하여 모델이 빠르고 정확하게 경제학적 문맥을 형성하도록 유도했다.

## 3. 모델 학습 결과 및 Loss 감소
결과 분석: 모델이 학습을 진행함에 따라 Loss(손실 함수) 값이 안정적으로 우하향하는 것을 확인했다. 이는 모델이 알파벳 배열의 통계적 규칙성을 성공적으로 파악하고 있음을 보여준다.

## 4. 생성된 텍스트 인사이트
1. 전문 어휘의 완벽한 모방: 모델은 단어 단위가 아닌 '글자 단위'로 학습했음에도 불구하고, transaction, money, labour, wages, exporters와 같은 경제학 핵심 어휘들을 완벽한 철자로 조합해 냈다.
2. 신조어 생성 (Character-level 학습의 증거): 생성된 텍스트 중간에 branking, merchances, cultivatoes 같은 사전적 의미가 없는 단어들이 관찰된다. 이는 모델이 기존 텍스트를 단순히 암기해서 복사하는 것이 아니라, 글자의 통계적 확률(예: 'merch' 뒤에 'ant' 대신 'ances'가 올 확률)을 스스로 계산하며 새로운 단어를 '창조'하고 있다는 것을 보여준다.
3. 문체 및 분위기 모방: 소설을 학습했을 때의 대화체는 사라지고, 길고 복잡한 18세기 학술 논문의 구조를 모방했다.

## 5. 결론
본 프로젝트를 통해, 동일한 구조의 인공지능 모델이라도 '어떤 성격의 데이터를 입력하느냐' 그리고 '어떻게 파라미터를 최적화하느냐'에 따라 그 결과물의 수준과 도메인이 완전히 달라진다는 것을 입증할 수 있었다.
