# AI-Finance-TinyGPT
## 1. 프로젝트 개요
본 프로젝트는 Character-level(글자단위) 기반의 소형 GPT 모델을 학습시켜 텍스트 생성의 원리를 이해하고 검증하는 것을 목표로 한다.
기본 제공된 소설 데이터셋을 넘어 애덤스미스의 『국부론 (The Wealth of Nations)』을 학습 데이터로 선정하여 인공지능이 전문적인 학술 어휘와 문체를 모방할 수 있는지 실험했다.
- 데이터 출처 : 프로젝트 구텐베르크 (Project Gutenberg)
- 데이터셋: 애덤 스미스, 『국부론 (The Wealth of Nations)』 순수 텍스트(Plain Text)

## 2. 하이퍼파라미터 튜닝 및 최적화 
국부론 데이터에 맞춰 모델의 성능을 극대화 하기 위해 코드 수정 작업을 진행하였다. 
- Epoch 증가 (TOTAL_EPOCHS = 2000): 복잡하고 긴 문장이 많은 경제학 서적의 특성을 반영하여, 모델의 데이터 패턴을 충분히 학습할 수 있도록 에포크를 2000회로 상향 조정했다.
- 초기 입력값 변경 (start_str = "market "): 학습 완료 후 텍스트를 생성할 때, 경제학 서적에 가장 빈번하게 등장하는 'market'을 시드 단어로 주입하여 모델이 빠르고 정확하게 경제학적 문맥을 형성하도록 유도했다.

## 3. 모델 학습 결과 및 Loss 감소
결과 분석: 모델이 학습을 진행함에 따라 Loss(손실 함수) 값이 안정적으로 우하향하는 것을 확인했다. 이는 모델이 알파벳 배열의 통계적 규칙성을 성공적으로 파악하고 있음을 보여준다.

## 4. 생성된 텍스트 인사이트
1. 전문 어휘의 완벽한 모방: 모델은 단어 단위가 아닌 '글자 단위'로 학습했음에도 불구하고, taxes, actual trade, proportion, capital exportation, foreign trade와 같은 경제학 핵심 어휘들을 완벽한 철자로 조합해 냈다.
2. 신조어 생성 (Character-level 학습의 증거): 생성된 텍스트 중간에 folliz, qualitizen, tobaccorden 같은 사전적 의미가 없는 단어들이 관찰된다. 이는 모델이 기존 텍스트를 단순히 암기해서 복사하는 것이 아니라, 글자의 통계적 확률을 스스로 계산하며 새로운 단어를 '창조'하고 있다는 것을 보여준다.
3. 문체 및 분위기 모방: 소설을 학습했을 때의 대화체는 사라지고, 길고 복잡한 18세기 학술 논문의 구조를 모방했다.

## 5. 결론
본 프로젝트를 통해, 동일한 구조의 인공지능 모델이라도 '어떤 성격의 데이터를 입력하느냐' 그리고 '어떻게 파라미터를 최적화하느냐'에 따라 그 결과물의 수준과 도메인이 완전히 달라진다는 것을 입증할 수 있었다.

## **Tiny GPT 모델의 주요 코드 개념
**토큰 및 위치 임베딩 (Token & Position Embedding)**
* 문자를 컴퓨터가 이해할 수 있는 숫자로 변환한다 (`stoi`, `itos`).
* `nn.Embedding`을 통해 단순한 숫자를 다차원 벡터 공간(128차원)으로 확장하여 문자 간의 의미적 유사성을 학습한다.
* RNN과 달리 모든 문자를 동시에 처리하는 GPT의 특성상, 문자의 순서 정보를 모델에 알려주기 위해 위치 임베딩(Position Embedding)을 더하여 입력값을 구성한다.


**마스크드 셀프-어텐션 (Masked Self-Attention)**
* 문장 내의 특정 문자가 다음 문자를 예측할 때, 앞서 등장한 어떤 문자에 집중(Attention)해야 하는지 가중치를 계산하는 핵심 레이어다.
* `Query(질의)`, `Key(키)`, `Value(값)` 행렬의 내적 연산을 통해 관계성을 파악한다.
* `torch.tril`을 사용한 마스킹(Masking) 기법을 적용하여, 과거의 문자들만 참조하고 미래의 문자를 미리 훔쳐보지 못하도록 제한한다.


**피드포워드 신경망 (FeedForward Network, FFN)**
* 어텐션 레이어에서 수집된 문맥 정보를 바탕으로, 각 위치별로 독립적인 비선형 변환을 수행한다.
* `Linear` 레이어와 `ReLU` 활성화 함수를 사용하여 모델이 데이터의 복잡한 패턴과 특징을 더 깊이 학습할 수 있도록 표현력(Capacity)을 확장한다.


**잔차 연결 및 레이어 정규화 (Residual Connection & Layer Normalization)**
* **잔차 연결 (`x = x + ...`):** 신경망이 깊어져도 학습 정보가 소실되지 않고 초기 정보가 끝까지 잘 전달되도록 돕는 지름길 역할을 한다.
* **레이어 정규화 (`nn.LayerNorm`):** 각 레이어를 통과할 때마다 값들의 분포를 안정적으로 정규화하여, 학습 속도를 높이고 모델의 안정성을 극대화한다.


**텍스트 생성 (Autoregressive Generation)**
* 학습된 모델에 초기 시드(Seed) 문자열(예: `"market "`)을 입력하면, 마지막 문자의 `Logits` 값을 기반으로 Softmax 함수를 통과시켜 다음 문자의 등장 확률을 계산한다.
* `torch.multinomial`을 이용해 확률에 기반한 무작위 추출을 수행함으로써, 매번 똑같은 글이 아닌 자연스럽고 다양한 문맥의 텍스트를 생성해 낸다.
